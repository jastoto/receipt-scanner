Google Cloud Setup

To use Google Drive integration, you need a free Google Cloud OAuth Client ID. Follow these steps:

  1. Go to console.cloud.google.com
  2. Create a new project (or select existing)
  3. Enable the Google Drive API (APIs & Services → Enable APIs)
  4. Go to APIs & Services → Credentials → Create Credentials → OAuth Client ID
  5. Choose Web Application
  6. Under Authorised JavaScript Origins, add your file URL or http://localhost
  7. Copy the Client ID and paste it below

Also enable Google Identity Services if prompted.

Keys are saved to localStorage in your browser only and never sent anywhere except the respective APIs.
